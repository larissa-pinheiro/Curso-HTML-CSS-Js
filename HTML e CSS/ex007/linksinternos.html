Usando links Internos

Faz a relação com outra página que está dentro do serviços e diz respeito à própria página

Esta é a primeira página do site. Se você quiser pode acessar também a minha segunda página.

Se eu colocar um link patrocinado ou link de uma empresa que eu não tenho/dou aval de colocar informações sobre ela lá, eu uso rel="nofollow" para que o Google não siga ou indexe meu site àquela página

Não esquecer de sempre que for colocar novas páginas internas no site, usar o atalho CTRL+SPACE para selecionar a página corretamente.

Se eu for usar uma outra pasta e eu quiser VOLTAR pra página inicial, eu digito ../index.html , esse ../ faz voltar a pasta.
