Nesta frase, temos um termo em destaque usando a tag <strong> (semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica)

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>.

Se eu quiser trocar a cor do marcador, basta utilizar esta formatação usando a informação style dentro da tag <mark style ="background-color: aquamarine">.

Caso eu queira mudar a cor de TODOS os marcadores da página, basta utilizar CSS lá no head, em style, criar a tag mark { background-color: aquamarine } por exemplo.
